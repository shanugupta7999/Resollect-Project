# React Dashboard Application 📊

A comprehensive and modular **React Dashboard Application** built with **ReactJS**, **Tailwind CSS**, and **React Icons**. The project is structured to ensure readability, reusability, and easy maintenance.

---

## 🔥 Features
- Modularized Code Structure for easy understanding.
- Dynamic **Loan Data Table** with sorting and filtering.
- **Sidebar Navigation** with different dashboard sections.
- **Document Upload Modal** for uploading files.
- Responsive UI with **Tailwind CSS** for a sleek design.

---



## 🛠️ Technologies Used
- **ReactJS**: Core library for UI development.
- **Tailwind CSS**: For responsive and modern styling.
- **React Icons**: For icons and illustrations.
- **JavaScript (ES6+)**: Programming logic.
- **JSON**: For data handling.

## 📁 **Folder and File Details**
- 📁 Frontend/
Contains the complete ReactJS frontend of the Resollect Project.

-  public/
Holds public static assets like index.html.
These files are accessible directly by the browser.

- 📁 src/
The main source directory for the React application.

- 📄 App.jsx
The root component managing routes and integrating all other components.

- 📄 index.jsx
The entry point of the React application.
Renders the root component (App.jsx) to the DOM.

- 📁 assets/
Stores images and other static assets used in the project.Example: ResollectIcon.png - Used as a logo or branding icon.

## 🚀 **Getting Started**
- Project Setup Instructions
To clone the repository and install dependencies, run the following commands:


- ## Clone the repositorygit clone https://github.com/shanugupta7999/Resollect-Project.git

- ## Navigate to the project directory 
    cd Resollect

- ## Install client dependencies
    cd Frontend
    npm install

- ## Start the client
    npm run dev
